Our project, 'Jum Kis Gali Ja Rahen Hain' provides a way for the user to figure out what's the best way to get from point A to point B in Habib.
The website is hosted locally and uses Flask to run. You will have to install it( pip install Flask)
Our folder also consists of a comparealgorithm file, which prints out the shortest path from point A to B but also gives a comparison between the algorithms used to find that point
